Use this service to update information about a company's share capital on the public register.

You can currently only use this service to file information about new shares that have been issued, also known as an allotment of shares.
How to file other share capital updates

You'll need to use the share capital forms for all other updates. Most can be filed online using the Upload a document to Companies House service.

You can only update shareholder information by filing a confirmation statement. You can do this online using our WebFiling service.

You'll need:

  • to sign in to or create a Companies House account, which is different from a WebFiling account
  • the company number
  • the company authentication code
